# Wildholz Pitch Deck

Ein interaktives Pitch-Deck für die Mediathek-Serie "Wildholz".

## Schnellstart (Offline & Lokal)

Um die Präsentation ohne Terminal-Befehle zu starten:

### Mac (Apple)
1. Doppelklick auf `start-mac.command`.
2. Die Präsentation öffnet sich automatisch in deinem Browser unter `http://localhost:3000`.

### Windows
1. Doppelklick auf `start-windows.bat`.
2. Die Präsentation öffnet sich automatisch in deinem Browser unter `http://localhost:3000`.

---

## Steuerung
- **Pfeiltasten (Links/Rechts)**: Vor/Zurück
- **Maus-Klick**: 
  - Rechte 70% vom Bildschirm: Weiter
  - Linke 30% vom Bildschirm: Zurück
- **Atmosphäre-Seite**: Die Punkte auf der rechten Seite sind interaktiv.
